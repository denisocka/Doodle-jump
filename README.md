# Doodle Jump - Learning Project

This is a simple **Doodle Jump** clone created as a learning project in C#.  
The game was developed to practice **Windows Forms**, **basic game logic**, and **object-oriented programming** concepts.

## How to Run

1. Download this repository.  
2. Open the solution file in Visual Studio.  
3. Build and run the project (`F5` or `Start` button).  
4. Use the **a or d** to move the character left and right. And use **left click on mouse or space** for attack
